Problem Statement:
Bob wants to send a secret code S to his boss.
He encrypts it using two keys: N and M.

Based on the OCR text and sample, the encryption formula
is interpreted as:
(((S^N % 10) ^ M) % 1000000007)

Write an algorithm to compute the encrypted code.

Input:
The first line contains an integer secretCode,
representing S.
The second line contains an integer firstKey,
representing N.
The third line contains an integer secondKey,
representing M.

Output:
Print an integer representing Bob's encrypted code.

Constraints:
1 <= secretCode <= 10^9
0 <= firstKey, secondKey <= 1000000007

Example:
Input:
2
3
4
