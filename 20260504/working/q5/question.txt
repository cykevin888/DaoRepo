Problem Statement:
Emerson likes reversing strings.
He is given two binary strings, str1 and str2.
He needs to convert str1 into str2 using these rules:

Step 1:
Reverse any one substring of length 2 in str1
to get a new string str1'.

Step 2:
Reverse any one substring of length 3 in str1'
to get a new string str1".

Step 3:
Reverse any one substring of length 4 in str1"
to get a new string str1'''.

Step 4, Step 5, and so on:
At step k, reverse any one substring of length (k + 1)
in the current string.

At each step, the chosen reversal must actually change
the string (i.e., new string != old string).

Find the minimum number of steps required to convert
str1 into str2.
If conversion is impossible, print -1.

Input:
The first line contains a binary string str1.
The second line contains a binary string str2.

Output:
Print an integer representing the minimum number
of steps required.
If no valid conversion exists, print -1.

Constraints:
2 <= N <= 30, where N is the string length.

Note:
At each step, Emerson can reverse only one substring.

中文翻译

题目描述：
Emerson 很喜欢反转字符串。
现在给他两个二进制字符串 `str1` 和 `str2`，
需要按如下规则把 `str1` 转换为 `str2`：

第 1 步：
在当前字符串中，选择一个长度为 2 的子串并反转，
得到新字符串。

第 2 步：
在新字符串中，选择一个长度为 3 的子串并反转，
得到下一状态字符串。

第 3 步：
选择一个长度为 4 的子串并反转。

第 4 步、第 5 步……依此类推：
第 k 步需要反转一个长度为 (k + 1) 的子串。

并且每一步都必须真正改变字符串
（即反转后不能与反转前相同）。

请计算把 `str1` 转为 `str2` 的最少步数。
如果无法转换，输出 `-1`。

输入：
第一行是二进制字符串 `str1`。
第二行是二进制字符串 `str2`。

输出：
输出最少步数；若无解输出 `-1`。

约束：
2 <= N <= 30，其中 N 为字符串长度。

说明：
每一步只能反转一个子串。
