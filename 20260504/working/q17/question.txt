The first line of the input consists of an integer size, representing the size of the

arr[N-1] representing the given list of numbers.

Output
Print N space-separated integers representing the list obtained by replacing the values
of the numbers with their corresponding index values in the list.

Constraints

Note
All the numbers in the list are unique.

Example

Input:
4
3201

Output:
2310

Explanation:
Before the change, the elements of the list are: arr[0]-3, arr[1]-2, arr[2]-0 and
arr[3]=1 After the change, the elements of the list are: arr[0]-2, arr[1]-3, arr[2]-1
and arr[3] -0

题目描述：
给定一个长度为 N 的数组，数组中的元素互不相同，且取值范围为 0 到 N-1。
请将数组中的每个值替换为“该值在原数组中的下标”。

输入：
第一行是数组长度 N。
第二行是 N 个整数。

输出：
输出替换后的数组。
