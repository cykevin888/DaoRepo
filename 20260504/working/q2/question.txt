Question (English)
Problem Statement:
You are given a list of integers and an integer K.
Write an algorithm to find the number of elements in
the list that are strictly less than K.

Input:
The first line contains an integer element_size,
representing the number of elements in the list (N).
The second line contains N space-separated integers:
element[1], element[2], ..., element[N].
The last line contains an integer num,
representing the value to compare with (K).

Output:
Print a positive integer representing the number
of elements in the list that are strictly less than num.

Constraints:
(Original text appears to have OCR issues in this section.
Please refer to the platform's official constraints if needed.)

Example:
Input:
7
1 7 4 5 6 3 2
5

Output:
4

Explanation:
The numbers less than 5 are 1, 4, 3, and 2.
So, the output is 4.
