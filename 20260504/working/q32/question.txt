Question
You are given a list of integers and an integer K. Write an algorithm to find the number
of elements in the list that are strictly less than K.

Input
The first line of the input consists of an integer K. The second line consists of N
space-separated integers. element[1], elemen t[2],..., element [N], representing the
list of integers. The last line consists of an integer n. n represents the integer to be
compared (K).

Output
Print a positive integer representing the number of elements in the list that are
strictly less than n

Constraints
105 ≤ n ≤ 109 •-10 element[1], element [2],..., element[ N]≤ 109

Example

Input:
7
17 AC472

题目描述：
给定一个整数列表和一个整数 K，统计列表中严格小于 K 的元素个数。

第一行表示元素个数 N。
第二行给出 N 个整数。

输出：
输出一个整数，表示严格小于 K 的元素数量。
