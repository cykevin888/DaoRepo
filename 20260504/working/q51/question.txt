Write an algorithm to find the number of occurrences of needle in given positive number
haystack.

Input
The first line of the input consists of an integer needle, representing a digit. The
second line consists of an integer haystack, representing the positive number.

Output
Print an integer representing the number of occurrences of needle in haystack.

Constraints
0s needle s9 0s haystacks 99999999

Example

Input:
2
123228

----------------------------------------
中文翻译
----------------------------------------

题目描述：
给定一个数字 needle 和一个正整数 haystack，统计 haystack 中数字 needle 出现的次数。

输入：
第一行是数字 needle。
第二行是正整数 haystack。

输出：
输出 needle 在 haystack 中出现的次数。
